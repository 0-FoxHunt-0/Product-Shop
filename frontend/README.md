# Frontend

React application for Product Shop with Chakra UI and Zustand state management.

## Features

- Product listing with responsive grid
- Create, update, delete products
- Modern UI with Chakra UI
- State management with Zustand

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- Chakra UI
- Zustand
- React Router
